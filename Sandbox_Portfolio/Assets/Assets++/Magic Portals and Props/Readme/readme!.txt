This asset is configured to be used with Universal Render Pipeline(URP), and LINEAR
color space,don't forget to configure it using these simple instruction: 

1. Import asset and Go to Edit-->Projects Setings-->Graphics
2. Click dot right to the Scriptable Render Pipeline Settings field
 and select UniversalRenderPipelineAsset from the list.
3.Go to Edit-->Projects Setings-->Player-->Other settings
And set Color space from gamma to Linear.
Done !
